  • What is the primary key for the Students table?: The primary key is id.
  • What are the two foreign keys in the Takes_Course table?: The two foreign keys are id and classid.
  • How are they foreign and primary keys related here?: Primary Keys are a unique value that represents a whole row. Foreign Keys are a type of key that exists in one table but refers to a primary key in another table allowing us to link tables together. The primary key id in the students table allows us to link it to the takes_courses table along with the primary key classid> in the courses table.
  • What is the schema of the courses table? The schema of the courses table are all the column headings (attributes) together. They are: classid, name, course_num, cred_hours.
