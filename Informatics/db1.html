Database 1

  • What are primary keys: Primary Keys are a unique value that represents a whole row. Every row or tuple will be differentiated by the primary key. An example of a primary key would be everyone’s student ID numbers at IU.
  • What are foreign keys: Foreign Keys are a type of key that exists in one table but refers to a primary key in another table allowing us to link tables together.
  • What is the difference between a tuple and an attribute: Each table is composed of two main elements: Attributes (Columns) and Tuples (Rows). A tuple is a row in a table, it is also known as a record. Its order does not matter. Attributes are column headings that describe the data of each column in a relational table. All the attributes together make up the “Schema” which is the description of the relation.
  • What is SQL? (What does it stand for and what does it do?): SQL stands for Structured Query Language. It is a bunch of short lines of code grouped together that executes some form of function or action on our databases. SQL is the standard language for Relational Database Management Systems.
